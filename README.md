# InspiraCred - Site Institucional

Site institucional da InspiraCred focado em crédito com garantia de imóvel (Home Equity).

## 🚀 Estrutura do Projeto

```
Infraestrutura/
│
├── index.html                    # Página principal (Simulação)
├── links/                        # Página de links (Link na bio)
│   └── index.html
│
├── assets/
│   ├── css/                      # Estilos
│   │   ├── main.css              # CSS da página principal
│   │   └── links.css             # CSS da página de links
│   │
│   ├── js/                       # JavaScript (futuro)
│   │
│   ├── img/                      # Imagens raster
│   │   ├── background.png
│   │   ├── landing-hero-desktop.png
│   │   └── ...
│   │
│   ├── icons/                    # Ícones SVG
│   │   ├── logo.svg
│   │   ├── chevron-right.svg
│   │   └── ...
│   │
│   └── fonts/                    # Fontes (futuro)
│
├── docs/                         # Documentação
├── robots.txt                    # Diretrizes para robôs de busca
├── sitemap.xml                   # Sitemap do site
└── README.md                     # Este arquivo
```

## 📋 Páginas

### Página Principal (`index.html`)
- Simulação de crédito com garantia de imóvel
- Informações sobre produtos Home Equity e Auto
- Depoimentos de clientes
- Benefícios e diferenciais

### Página de Links (`links/`)
- Links para redes sociais
- Links para WhatsApp
- Acesso rápido aos produtos
- Substituto profissional para Linktree

## 🎨 Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com variáveis CSS
- **JavaScript** - (quando necessário)
- **Google Fonts** - Montserrat, Inter, Instrument Sans

## 🛠️ Como Usar

### Desenvolvimento Local

Abra o arquivo `index.html` diretamente no navegador:

```bash
# No diretório Infraestrutura/
start index.html  # Windows
open index.html   # macOS
xdg-open index.html # Linux
```
